# 🎬 RajStream

RajStream is a responsive OTT-style movie streaming web application inspired by modern streaming platforms like Netflix.

## 🚀 Features

- 🎥 Video playback functionality
- 🔥 Trending, Bollywood, Hollywood & Anime sections
- 🖱 Interactive movie cards with hover effects
- 📱 Fully responsive design (Mobile + Desktop)
- 🌙 Clean dark theme UI

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
